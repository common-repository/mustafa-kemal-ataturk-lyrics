== Kurulum ==

1) İndirdiğiniz "mustafa-kemal-ataturk-lyrics.zip" dosyasını açıp sitenizin "plugins" dizini altına "mustafa-kemal-ataturk-lyrics" klasörü içerisinde atın. 
Olağan şartlarda eklentinin dizin yolu şu şekilde görünmelidir:
			wp-content\plugins\mustafa-kemal-ataturk-lyrics\[...dosyalar...]
			
2) Admin panelinde eklentiler bölümünden etkinleştirdiğiniz anda çalışmaya başlayacaktır. 

== Sözcük ekleme == 

1) Eklentinin klasöründe bulunan "mka_lyrics_all.txt" dosyasının içeriğine yeni sözler ekleyebilirsiniz. Her satırın sonuna "#EOL" belirtecini koymalısınız. Aksi halde eklenti düzgün çalışmayacaktır.
